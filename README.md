# iot-python-2026
IoT개발자 파이썬 리포지토리






- 문법 비교표

|이론개념|C/C++|Python|
|--------|-----|------|
|출력|print(), cout|`print()`|
|변수 선언|int a = 10;|`a = 10`|
|조건문|if (a > b) { ... }|`if a > b:`|
|반복문|for (int i = 0; i < 10; i++) { }|`for i in range(10):`|
|함수|int add(int a, int b) { }|`def add(a, b):`|
|배열|int arr[5]|`list`|
|문자/문자열|char, char[], char*, string|`str`|

- 장점

- 들여쓰기가 코드 블록, {} 불필요
- 선언이 없음
- 리스트가 배열보다 훨씬 편하고 간결
- 문자열 처리 간단
- 함수 만들기 간단
